Top 5 Best-Selling ProductsTop 5 najprodavanijih proizvoda

SELECT p.post_title AS Product_Name, SUM(oi.order_item_qty) AS Total_Sold
FROM wp_posts p
JOIN wp_woocommerce_order_itemmeta oim ON oi.order_item_id = oim.order_item_id
JOIN wp_woocommerce_order_items oi ON p.ID = oi.order_item_id
WHERE p.post_type = 'product' AND oim.meta_key = '_qty'
GROUP BY p.ID
ORDER BY Total_Sold DESC
LIMIT 5;

Retrieves top-selling products for analytics.

Dohvata najprodavanije proizvode za analitiku.

Recent User ActivityNedavna aktivnost korisnika

SELECT u.user_login, um.meta_value AS Last_Login
FROM wp_users u
LEFT JOIN wp_usermeta um ON u.ID = um.user_id
WHERE um.meta_key = 'last_login'
AND um.meta_value > DATE_SUB(NOW(), INTERVAL 30 DAY)
ORDER BY um.meta_value DESC;

Queries recent user logins for activity reports.

Upit za nedavne logine korisnika za izveštaje.

Database OptimizationOptimizacija baze podataka

DELETE FROM wp_postmeta
WHERE meta_key IN ('_edit_lock', '_edit_last')
AND post_id NOT IN (SELECT ID FROM wp_posts);
CREATE INDEX idx_postmeta_postid ON wp_postmeta(post_id);

Cleans orphaned postmeta and adds an index for faster queries.

Čisti napuštene postmeta podatke i dodaje indeks za brže upite.
